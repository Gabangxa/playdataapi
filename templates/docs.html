API Reference

PlayDataAPI gives indie mobile developers access to Google Play app metadata, reviews, search, and trending charts via a simple REST API.

Free tier · 100 requests/day · No signup required

Quick Start

curl https://your-app.replit.app/apps/com.spotify.music \ -H "Authorization: Bearer demo-key-123"

Authentication

All API requests (except /health) require an API key passed as a Bearer token in the Authorization header.

Authorization: Bearer demo-key-123

App Details

GET /apps/{package_id}

Fetch full app metadata for any Google Play package ID. Results are cached for 1 hour.

Path Parameters

NameTypeDescription
package_id requiredstringGoogle Play package ID, e.g. com.spotify.music

Query Parameters

NameDefaultDescription
langenLanguage code (ISO 639-1)
countryusCountry code (ISO 3166-1 alpha-2)

Example Request

curl https://your-app.replit.app/apps/com.spotify.music \ -H "Authorization: Bearer demo-key-123"

Example Response

{ "package_id": "com.spotify.music", "title": "Spotify: Music and Podcasts", "developer": "Spotify AB", "developer_id": "Spotify+AB", "category": "MUSIC_AND_AUDIO", "rating": 4.3, "ratings_count": 32000000, "installs": "1,000,000,000+", "price": 0.0, "free": true, "description": "...", "last_updated": "2026-03-10", "version": "8.9.90.432", "android_version": "Varies with device", "icon_url": "https://..." }

App Reviews

GET /apps/{package_id}/reviews

Fetch recent user reviews from Google Play. Results cached for 30 minutes.

Query Parameters

NameDefaultDescription
count20Number of reviews to return (max 100)
sortnewestnewest or most_relevant
langenLanguage code
countryusCountry code

Example Request

curl "https://your-app.replit.app/apps/com.spotify.music/reviews?count=5&sort=newest" \ -H "Authorization: Bearer demo-key-123"

Example Response

{ "package_id": "com.spotify.music", "count": 5, "reviews": [ { "review_id": "abc123", "author": "John D.", "rating": 5, "text": "Great app!", "date": "2026-03-20", "thumbs_up": 124, "reply": null } ] }

Search

GET /search

Search Google Play by keyword.

Query Parameters

NameDefaultDescription
q required—Search keyword or phrase
n20Number of results (max 50)
langenLanguage code
countryusCountry code

Example Request

curl "https://your-app.replit.app/search?q=music+player&n=5" \ -H "Authorization: Bearer demo-key-123"

Example Response

{ "query": "music player", "count": 5, "results": [ { "package_id": "com.spotify.music", "title": "Spotify", "developer": "Spotify AB", "rating": 4.3, "installs": "1,000,000,000+", "icon_url": "https://..." } ] }

Trending / Top Charts

GET /trending

Fetch Google Play top charts by collection and optional category.

Query Parameters

NameDefaultDescription
collectiontop_freetop_free | top_paid | top_grossing
category—e.g. MUSIC_AND_AUDIO, GAMES, SOCIAL
n20Number of results (max 100)
langenLanguage code
countryusCountry code

Example Request

curl "https://your-app.replit.app/trending?collection=top_free&category=MUSIC_AND_AUDIO&n=5" \ -H "Authorization: Bearer demo-key-123"

Example Response

{ "collection": "top_free", "category": "MUSIC_AND_AUDIO", "count": 5, "apps": [ { "rank": 1, "package_id": "com.spotify.music", "title": "Spotify", "developer": "Spotify AB", "rating": 4.3, "installs": "1,000,000,000+", "icon_url": "https://..." } ] }

Health Check

{"status": "ok"}

Error Codes

All errors use a consistent JSON format:

{"error": "Human-readable message", "code": "machine_readable_code"}
HTTP StatusCodeMeaning
400invalid_paramMissing or invalid query parameter
401auth_requiredMissing or invalid API key
404app_not_foundPackage ID not found on Google Play
429rate_limitDaily or per-second rate limit exceeded
502upstream_errorFailed to fetch data from Google Play
